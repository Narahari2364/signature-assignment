# 1947 Indian Restaurant Website

A responsive website for the 1947 Indian Restaurant, celebrating India's rich culinary heritage since independence.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop devices
- **Navigation**: Working navigation links across all pages
- **Email Links**: Functional email links in footer
- **Image Links**: External image links to community partners
- **Menu Content**: Comprehensive menu with categories and pricing
- **Contact Forms**: Reservation and contact forms
- **Modern CSS**: Flexbox and Grid layouts with media queries

## Pages

1. **Homepage (index.html)**: Hero section, featured categories, restaurant highlights
2. **Menu (menu.html)**: Complete menu with categories (Nibbles, Fancy Feasts, Treats, Drinks)
3. **About Us (about.html)**: Restaurant story, values, chef information, awards
4. **Reservations (reservations.html)**: Reservation form, location map, private events
5. **Contact (contact.html)**: Contact form, location & hours, social media
